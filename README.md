# Weather Dashboard

In this application, you can know the current weather data of any location. [Live Site](https://weather-dashboard-2-sable.vercel.app/)

## Features:
1. The app asks for location permission and shows the weather data of user's location.
2. User can search for weather data of other regions, too.
3. User can favorite or unfavorite a location and then find it in the Favorite modal and quickly bring the weather data of favorite regions by just clicking the locations.
4. The background image changes based on the weather type.
5. The app shows error on invalid location search

## Technologies used:
1. APIs: Open Weather Map API to fetch weather data and free Geocoding API to fetch the geocoordinates of a location
2. React. Implemented builtin hooks e.g. useState, useEffect and custom hooks, context API etc.
